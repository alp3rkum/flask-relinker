Flask Relinker Algorithm

The Flask Relinker Custom Algorithm is an URL shortening algorithm written in Python (as this application uses Flask for its back-end) to shorten your URLs with improved safety and convenience. This algorithm is the heart of the Flask Relinker, and is reserved for the members of the website.

Features

  • SHA-384 Hashing Algorithm: While SHA-256 is a reliable hashing algorithm, Flask Relinker Custom Algorithm ups the ante a little bit more and makes use of SHA-384 to ensure improved safety and prevention of collisions. Though it might not appear any different in your searchbar, the URL is specifically designed to run through Flask Relinker.
  • Link Saving Feature: Reserved for members only, the feature to save your favorite links is also present in Flask Relinker. Free members can save up to 10 links.

How Does an URL Shortener Run?

While it appears a bit mysterious, an URL Shortener application actually has a simple principle. The shortened URL is a route that is accessible through the application's backend. Using this route, the application redirects you to the respective page (if the application has any) and then redirects you to the actual URL. Nothing truly magical, but nothing too ordinary either.
